Calculadora: Crea una calculadora básica utilizando HTML, CSS y JavaScript. Para ello, 
deberás utilizar el DOM para manipular los botones y realizar las operaciones matemáticas 
con JavaScript.

Descripción: El objetivo de este ejercicio es crear una calculadora básica utilizando HTML, 
CSS y JavaScript. La calculadora debe ser capaz de realizar operaciones matemáticas básicas 
como suma, resta, multiplicación y división.

Requerimientos Funcionales:

∆ La calculadora debe tener un display donde se muestre el resultado de las operaciones matemáticas.
∆ Debe haber botones para cada uno de los dígitos del 0 al 9.
∆ Debe haber botones para cada una de las operaciones matemáticas básicas: suma (+), resta (-), multiplicación (*), y división (/).
∆ La calculadora debe ser capaz de realizar operaciones matemáticas en tiempo real al presionar los botones correspondientes.
∆ Debe haber un botón para borrar el último dígito ingresado y otro para borrar todo el contenido del display.

Requerimientos No Funcionales:

∆ La interfaz de la calculadora debe ser intuitiva y fácil de usar.
∆ La calculadora debe ser compatible con diferentes navegadores web y dispositivos.
∆ El código debe ser limpio, organizado y fácil de mantener.
∆ La calculadora debe ser responsiva y ajustarse automáticamente a diferentes tamaños de pantalla.
∆ La calculadora debe ser accesible para personas con discapacidades visuales o de movilidad.
